ED Simulation
=============

1. Create a Python venv and install requirements:
   
   Refer to the README in the root directory

2. Run the simulation:
   python main.py

   This creates the following files in output folder 'patient_records.csv', 'queue_records.csv', 'ed_summary.csv'.

3. To experiment:
   Edit ed_sim_settings.json to control parameters passed into the simulation
